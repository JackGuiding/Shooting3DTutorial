==== FPS ====
FPS (First Person Shoot)
- 看不见自己

==== TPS ====
TPS (Third Person Shoot)
看到自己的头(全身)

==== Generic ====
- 子弹飞出视线范围就销毁
- 准心: UI 实现, 屏幕正中心
- 子弹类型:
    1. (现代)实弹: 子弹本身也是通过射线检测
    2. (20年前旧设备的情况)虚弹(没有任何实体产生): 通过物理射线检测
- 子弹飞行方向:
    1. 从相机的坐标向相机的正前方(准心)
    2. 子弹是往角色面向(枪口面向)的方向发射
- 隐藏鼠标指针: UnityEngine.Cursor 类
- 逻辑表现分离:
    - 子弹看上去飞行的方向 是从枪口发出
    - 逻辑上飞行的方向是从相机发出
